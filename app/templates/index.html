API CHU

Intructions de navigation

  • /materiel : GET la table matériel
  • /materiel/'id' : GET élément de la table matériel par id
  • /employe : GET la table employé_informatique
  • /employe/'id' : GET élément de la table employé_informatique par id

Les méthodes POST et DELETE sont également disponibles via Postman.

Pour les POST, les id sont en auto-incrémentation donc déjà gérés.

Pour DELETE, il suffit d'indiquer l'id de l'élément que l'on souhaite supprimer
